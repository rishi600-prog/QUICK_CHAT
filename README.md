#  QUICK_CHAT

QUICK_CHAT is a real-time, one-on-one messaging web application. Built with full-stack technologies and web sockets.

---

##  Tech Stack

**Frontend:**
- React.js
- HTML
- Javascript
- Tailwind CSS / CSS Modules
- Axios

**Backend:**
- Node.js
- Express.js
- Socket.IO

**Database:**
- MongoDB
- Mongoose

**Authentication:**
- JWT (JSON Web Tokens)
- Bcrypt.js
